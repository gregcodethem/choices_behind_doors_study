You will now have two practise turns on the memory task.

You will see a matrix with randomly placed dots on it.

You will have only 1 second to memorise the position of the dots on the matrix.

After this 1 second memorisation period, you will see an empty matrix to place the dots as well as you can remember. You need to click inside the each square to place the dots. If you think that you place any dot inside a wrong square, you can remove the dot by clicking on it. There is no time limit while placing the dots on the matrix.


This is a trial task. It is crucial that the dot pattern is reproduced correctly in the upcoming Monty Hall Game.


When you are ready, go to the next page. You will see a countdown from 5 before starting the practice memory task. When the countdown finishes you will see a fixation cross (+) on the centre of the screen for 2 seconds before the practice memory task starts.
