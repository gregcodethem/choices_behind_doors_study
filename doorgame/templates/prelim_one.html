Welcome to the Monty Hall Game with the memory task.

This experiment was prepared to study people's decision making strategies under a memory condition

The memory condition contains a matrix with the randomly positioned dots.

Before the Monty Hall Game, you will see the matrix with randomly placed dots on it for 1s

Then you will play the Monty Hall Game. After your decision is recorded, an empty matrix will be presented. You will be asked to place the dots on the matrix correctly.

There is no time limit for the experiment
